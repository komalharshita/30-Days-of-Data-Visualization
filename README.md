# 30-Days-of-Data-Visualization
A structured 30-day hands-on journey mastering Data Visualization using Python, Seaborn, and Matplotlib. Includes guided notebooks, EDA workflows, and interview-focused analytical thinking.

    https://github.com/komalharshita/30-Days-of-Data-Visualization.git

2.  Navigate into the folder: cd 30-Days-of-Data-Visualization

3.  Launch Jupyter: jupyter notebook

4.  Open any Day_X notebook and explore.
